Stash It

Welcome to Stash It!

Your personal stash for code snippets and interesting links

How to stash stuff:

  1. Stash code snippets: Select JSON/XML text on any webpage, right-click and choose "Stash selected snippet"
    or press Ctrl+Shift+L (Cmd+Shift+L on Mac)
  2. Stash links: Found something worth revisiting? Press
    Ctrl+Shift+U (Cmd+Shift+U on Mac)
  3. View your stash - snippets get formatted with pretty syntax highlighting
  4. Browse history in the sidebar to dig through your collection
Tip: Think of it as your code snippet treasure chest
